##### Models That Explain Themselves: Multimodal Task with Prompting Large Language Models #####

##### Project description #####
	  
	  In this project we perform a multimodal task which involves first captioning to obtain
	  textual representation of an image, then pass that as context to a LLM (GPT)
	  
	  The code was run on Ubuntu 22.04


##### Contents #####

      1. Installation

         The project assumes a working installation of python3. Steps are as follows:
	         - Download dataset from link provided in the notebook.
			   - https://mega.nz/folder/0SQCmJzZ#a51rhBc4zbfe6VwGKqyrQA
	         - Create a conda environment by running the env.yml
	         	$ conda env create -f f"{path_to_files}\env.yml"
	         - Create a kernel that can be used to run jupyter notebook commands inside the virtual environment
	         	$ python -m ipykernel install --user --name awaKE_4 --display-name "awaKE_4"
	         - Open jupyter notebook
	         - Open awaKE_4 and change the kernel to "awaKE_4"
